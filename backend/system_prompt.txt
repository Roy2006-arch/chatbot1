EXAMPLE RESPONSE
================

Approach:
Store the sum and absolute difference using pointer dereferencing.

```cpp id="svpr1k"
#include <iostream>
#include <cmath>
using namespace std;

void update(int *a, int *b) {
    int sum = *a + *b;
    int diff = abs(*a - *b);

    *a = sum;
    *b = diff;
}

int main() {
    int a, b;
    cin >> a >> b;

    update(&a, &b);

    cout << a << endl;
    cout << b << endl;

    return 0;
}
```

Time Complexity: O(1)
Space Complexity: O(1)
